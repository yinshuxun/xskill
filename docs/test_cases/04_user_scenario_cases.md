## 1. 技能创建与管理场景

### Case ID: `TC-E2E-001`
- **场景描述**: 创建 Skill 并验证 Hub 生成
- **用户角色**: 开发者
- **前置条件**: 
  - 应用已启动
  - 用户已登录
- **测试步骤**:
  1. 点击 Hub 页面的 "New Skill" 按钮
  2. 填写表单:
     - Name: "my-test-skill"
     - Description: "A test skill for E2E testing"
     - Content: "This skill does something useful"
  3. 点击 "Create Skill" 按钮
  4. 等待创建完成
  5. 验证技能出现在 Hub 列表
  6. 验证文件系统中生成 `~/.xskill/skills/my-test-skill/`
  7. 验证目录结构: `SKILL.md`, `scripts/`, `references/`, `assets/`
  8. 验证 `SKILL.md` 内容正确
- **预期结果**: 
  - 技能成功创建
  - 文件系统正确生成
  - UI 正确显示
- **验证点**:
  - 表单验证生效
  - 后端处理正确
  - 文件系统操作正确
  - UI 反馈正确
