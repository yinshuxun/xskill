### 📊 文档分类

#### 需求文档 (`docs/requirements/`)
- **愿景与指南**: 项目愿景、核心价值、使用指南
- **产品需求**: 功能需求、非功能需求、用户故事
- **架构设计**: 系统架构、技术选型、数据流
- **竞品分析**: 市场分析、竞品对比、差异化
- **完整需求**: 详细的交互流程、API 文档、UI/UX 规范

#### 测试用例 (`docs/test_cases/`)
- **前端测试**: React 组件测试、交互测试、E2E 测试
- **后端单元测试**: Rust 函数测试、算法测试
- **后端集成测试**: 端到端业务流程测试
- **用户场景测试**: 完整用户旅程测试
- **测试模板**: 测试用例编写模板和规范

#### 开发文档 (`docs/development/`)
- **CLI 指南**: 命令行工具使用说明
- **API 文档**: 后端 API 接口文档
- **开发指南**: 开发环境搭建、代码规范
- **部署指南**: 构建和部署流程
